# 程序员越老越值钱的技能

>  研究程序员职业生涯中与年龄正相关的技能——有哪些能力不会因体力下降而贬值，反而随经验增长持续升值，以及如何系统地培养这些能力。

## 🎯 研究动机

程序员是一个技术迭代极快的职业，"35 岁危机"是行业内反复讨论的话题。但现实中确实存在一批越老越吃香的程序员。他们的价值不在于拼体力、拼加班，而在于积累了一种 **"复利型"技能组合**——本项目的目标就是把这种组合结构化地梳理出来。

## 📋 研究内容

目前包含一篇完整研究报告：[`程序员越老越值钱的技能-研究报告.md`](./程序员越老越值钱的技能-研究报告.md)，覆盖以下维度：

1. **与年龄正相关的技能类别**（不因体力下降而贬值）
2. **每个类别下的具体技能清单**
3. **每类技能的培养路径与方法**
4. **国内外大龄程序员的真实案例**

>  报告核心框架：**技能折旧速度决定年龄相关性**。底层原理和领域知识的半衰期以十年为单位，框架工具仅 1-3 年。这是一场"学习 vs 折旧"的赛跑——技能陈旧而非年龄本身是薪资差距的主因。详见报告第九章参考文献。

## ⚡ 报告核心结论速览

| 技能类别                  | 核心能力                       | 培养方式                       |
| --------------------- | -------------------------- | -------------------------- |
| 🏗️ **架构与系统设计**       | 复杂系统分解、非功能性权衡              | 精读经典系统架构、参与大型项目            |
| 🏭 **领域知识纵深**         | 垂直行业 deep domain knowledge | 长期深耕一个行业，积累不可替代的业务理解       |
| 🔍 **调试与排障直觉**        | 模式匹配式快速定位                  | 大量真实故障排查经验，建立 mental model |
| 🎓 **技术判断力**          | 技术选型、避免过度设计                | 经历过多次技术周期的兴衰               |
| 🗣️ **沟通与 Mentoring** | 把复杂问题讲清楚、带人成长              | 主动承担技术分享、文档化、带新人           |
| 🔗 **跨领域整合**          | AI+行业、技术+商业                | 构建个人复利资产（开源/博客/SaaS）       |
